# 项目简介

## 核心功能
- 支持660张标注数据集，包含['floor', 'wall']两类目标
- 提供完整的源码、训练好的模型及训练结果
- 支持图片、视频及摄像头进行检测
- 支持批量检测功能，界面可实时显示检测结果
- 提供UI界面，可选择显示标签和原图
- 支持自定义置信度和IOU阈值
- 支持选择不同检测模型
- 可查看批量检测的每一张结果

## 适用场景
- 建筑装修行业中墙面与地板的自动化检测
- 室内设计领域的墙面地板状态评估
- 房地产验收中的墙面地板质量检测
- 智能家居环境中的墙面地板识别分析

## 技术特点
- 基于YOLOv8实例分割算法
- 采用Pycharm/VSCode开发环境
- 集成PyQt5用户界面
- 支持Python 3.9及相关依赖库
- 提供完整的部署环境和运行教程
